# Demo Script – Conference Room Booking System

## Demo Objective
Demonstrate that a new developer can run and interact with the system using provided documentation.

## Step 1 – Open Repository
- Open the GitHub repository
- Show README.md
- Highlight setup and prerequisites

## Step 2 – Start the Application
- Run `docker-compose up`
- Explain which services are starting

## Step 3 – Open API Documentation
- Navigate to Swagger/OpenAPI endpoint
- Explain available endpoints

## Step 4 – Create a Booking
- Use Swagger or curl to create a booking
- Show request and response

## Step 5 – Verify Booking
- Fetch bookings via GET request
- Confirm data persistence

## Demo Outcome
- System runs locally
- Documentation is accurate
- API functions as expected
